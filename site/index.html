Flex Grid Framework

Because flexbox and mathematics are fun :D

View a demo Download V 0.1.0

Note: To use the framework, you'll need to compile your style with the Stylus preprocessor using NodeJS, for more information see NodeJS and Stylus.

The basics

Create you markup


<div class="container">
  <div class="yellow-bar"></div>
  <div class="yellow-bar"></div>
</div>

Import the framework


@import "flexgridframework"

Set your rows and columns


.container
  row()

.yellow-bar
  col(6)

.yellow-bar
  col(6)
